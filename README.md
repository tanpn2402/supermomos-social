# #supermomos-social

The webpage that allows users to create/host a new event/social.


## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

NextJS dev server listens on port `3000`


## Powered by

2. TailwindCSS.
